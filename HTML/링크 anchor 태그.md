## Anchor 태그
<br/> 굉장히 자주 사용 하는 태그이며 **href**는 태그 추가적인 정보를 주는 속성입니다<br/>
```html
<a href="주소"> 링크 </a>
```
```html
<a href="https://www.google.com"> 구글 사이트 이동 </a>
```
여기서 **target_blank** 속성을 넣게되면 **새탭**으로 이동하며 사이트가 열리게 됩니다<br/>
```html
<a href="https://www.google.com" target="_blank" > 구글 사이트 이동 </a> 
```
